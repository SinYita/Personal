Information Theory Basics: Entropy and Mutual Information

Information theory, founded by Claude Shannon in 1948, is the theoretical cornerstone of modern communication, compression algorithms, and machine learning.

1. Shannon Entropy

For a discrete random variable XXX with probability distribution P(X=xi)=piP(X = x_i) = p_iP(X=xi​)=pi​, Shannon entropy is defined as:

H(X)=−∑ipilog⁡2piH(X) = -\sum_{i} p_i \log_2 p_iH(X)=−∑i​pi​log2​pi​

Intuition: Entropy measures the uncertainty of a random variable. The entropy is maximized when the distribution is uniform, and is 0 for a deterministic event.

Example: The binary entropy function for a binary random variable:

Hb(p)=−plog⁡2p−(1−p)log⁡2(1−p),p∈[0,1]H_b(p) = -p \log_2 p - (1-p) \log_2(1-p), \quad p \in [0,1]Hb​(p)=−plog2​p−(1−p)log2​(1−p),p∈[0,1]

When p=0.5p = 0.5p=0.5, Hb(0.5)=1H_b(0.5) = 1Hb​(0.5)=1 bit, the uncertainty is maximized.

2. Joint Entropy and Conditional Entropy

Joint Entropy:

H(X,Y)=−∑x,yp(x,y)log⁡p(x,y)H(X, Y) = -\sum_{x,y} p(x,y) \log p(x,y)H(X,Y)=−∑x,y​p(x,y)logp(x,y)

Conditional Entropy:

H(Y∣X)=∑xp(x)H(Y∣X=x)=−∑x,yp(x,y)log⁡p(y∣x)H(Y \mid X) = \sum_x p(x) H(Y \mid X=x) = -\sum_{x,y} p(x,y) \log p(y \mid x)H(Y∣X)=∑x​p(x)H(Y∣X=x)=−∑x,y​p(x,y)logp(y∣x)

Chain Rule:

H(X,Y)=H(X)+H(Y∣X)=H(Y)+H(X∣Y)H(X, Y) = H(X) + H(Y \mid X) = H(Y) + H(X \mid Y)H(X,Y)=H(X)+H(Y∣X)=H(Y)+H(X∣Y)

3. Mutual Information

Mutual information measures the degree of dependency between two random variables:

I(X;Y)=H(X)−H(X∣Y)=H(Y)−H(Y∣X)I(X; Y) = H(X) - H(X \mid Y) = H(Y) - H(Y \mid X)I(X;Y)=H(X)−H(X∣Y)=H(Y)−H(Y∣X)

Equivalently:

I(X;Y)=∑x,yp(x,y)log⁡p(x,y)p(x)p(y)I(X; Y) = \sum_{x,y} p(x,y) \log \frac{p(x,y)}{p(x)p(y)}I(X;Y)=∑x,y​p(x,y)logp(x)p(y)p(x,y)​

Properties:

  • I(X;Y)≥0I(X; Y) \geq 0I(X;Y)≥0, with equality if and only if XXX and YYY are independent.
  • I(X;Y)=I(Y;X)I(X; Y) = I(Y; X)I(X;Y)=I(Y;X) (Symmetry).
  • I(X;X)=H(X)I(X; X) = H(X)I(X;X)=H(X) (Self-information equals entropy).

4. KL Divergence

KL divergence (relative entropy) measures the difference between two distributions:

DKL(P∥Q)=∑xp(x)log⁡p(x)q(x)D_{KL}(P \| Q) = \sum_x p(x) \log \frac{p(x)}{q(x)}DKL​(P∥Q)=∑x​p(x)logq(x)p(x)​

Properties: DKL(P∥Q)≥0D_{KL}(P \| Q) \geq 0DKL​(P∥Q)≥0, and DKL(P∥Q)=0  ⟺  P=QD_{KL}(P \| Q) = 0 \iff P = QDKL​(P∥Q)=0⟺P=Q.

Relationship between mutual information and KL divergence:

I(X;Y)=DKL(p(x,y)∥p(x)p(y))I(X; Y) = D_{KL}(p(x,y) \| p(x)p(y))I(X;Y)=DKL​(p(x,y)∥p(x)p(y))

5. Applications in Machine Learning

Cross-Entropy Loss

Cross-entropy is commonly used in classification tasks:

H(p,q)=−∑xp(x)log⁡q(x)=H(p)+DKL(p∥q)H(p, q) = -\sum_x p(x) \log q(x) = H(p) + D_{KL}(p \| q)H(p,q)=−∑x​p(x)logq(x)=H(p)+DKL​(p∥q)

Minimizing cross-entropy is equivalent to minimizing KL divergence, i.e., making the model distribution qqq approximate the true distribution ppp.

Information Gain in Decision Trees

When selecting feature AAA, the information gain is:

Gain(S,A)=H(S)−∑v∣Sv∣∣S∣H(Sv)=I(S;A)\text{Gain}(S, A) = H(S) - \sum_{v} \frac{|S_v|}{|S|} H(S_v) = I(S; A)Gain(S,A)=H(S)−∑v​∣S∣∣Sv​∣​H(Sv​)=I(S;A)

import numpy as np

def entropy(labels):
    n = len(labels)
    counts = np.bincount(labels)
    probs = counts / n
    return -np.sum(p * np.log2(p + 1e-12) for p in probs if p > 0)

def information_gain(parent_labels, splits):
    parent_entropy = entropy(parent_labels)
    n = len(parent_labels)
    weighted = sum(len(s) / n * entropy(s) for s in splits)
    return parent_entropy - weighted
