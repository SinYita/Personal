Deriving Backpropagation with LaTeX

Backpropagation is the core algorithm for training neural networks. This article provides a rigorous mathematical derivation to help you understand its essence.

1. Basic Symbol Definitions

Assume a neural network has LLL layers. The weight matrix for layer lll is W(l)W^{(l)}W(l), the bias vector is b(l)b^{(l)}b(l), and the activation function is σ\sigmaσ.

The pre-activation for layer lll:

z(l)=W(l)a(l−1)+b(l)z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}z(l)=W(l)a(l−1)+b(l)

Activation output:

a(l)=σ(z(l))a^{(l)} = \sigma(z^{(l)})a(l)=σ(z(l))

2. Loss Function

For Mean Squared Error (MSE) loss:

L=12∣a(L)−y∣2\mathcal{L} = \frac{1}{2} | a^{(L)} - y |^2L=21​∣a(L)−y∣2

3. Chain Rule and Gradient Calculation

Define the error signal:

δ(l)=∂L∂z(l)\delta^{(l)} = \frac{\partial \mathcal{L}}{\partial z^{(l)}}δ(l)=∂z(l)∂L​

Output layer error:

δ(L)=(a(L)−y)⊙σ′(z(L))\delta^{(L)} = (a^{(L)} - y) \odot \sigma'(z^{(L)})δ(L)=(a(L)−y)⊙σ′(z(L))

Where ⊙\odot⊙ denotes the Hadamard product (element-wise multiplication).

Hidden layer error (backward propagation):

δ(l)=(W(l+1)⊤δ(l+1))⊙σ′(z(l))\delta^{(l)} = \left( W^{(l+1)\top} \delta^{(l+1)} \right) \odot \sigma'(z^{(l)})δ(l)=(W(l+1)⊤δ(l+1))⊙σ′(z(l))

4. Parameter Gradients

∂L∂W(l)=δ(l)(a(l−1))⊤\frac{\partial \mathcal{L}}{\partial W^{(l)}} = \delta^{(l)} \left( a^{(l-1)} \right)^\top∂W(l)∂L​=δ(l)(a(l−1))⊤

∂L∂b(l)=δ(l)\frac{\partial \mathcal{L}}{\partial b^{(l)}} = \delta^{(l)}∂b(l)∂L​=δ(l)

5. Gradient Descent Update

W(l)←W(l)−η∂L∂W(l)W^{(l)} \leftarrow W^{(l)} - \eta \frac{\partial \mathcal{L}}{\partial W^{(l)}}W(l)←W(l)−η∂W(l)∂L​

b(l)←b(l)−η∂L∂b(l)b^{(l)} \leftarrow b^{(l)} - \eta \frac{\partial \mathcal{L}}{\partial b^{(l)}}b(l)←b(l)−η∂b(l)∂L​

Where η\etaη is the learning rate.

6. Complete Algorithm Flow

  1. Forward Propagation: Compute z(l)z^{(l)}z(l) and a(l)a^{(l)}a(l) for each layer.
  2. Compute Loss: L(a(L),y)\mathcal{L}(a^{(L)}, y)L(a(L),y)
  3. Backpropagation: Compute δ(l)\delta^{(l)}δ(l) from l=Ll = Ll=L down to l=1l = 1l=1.
  4. Update Parameters: Use the computed gradients to update W(l)W^{(l)}W(l) and b(l)b^{(l)}b(l).

Summary

Backpropagation is essentially an efficient implementation of the chain rule. Understanding its mathematical foundation helps us better debug networks, choose activation functions, and design optimization strategies.
